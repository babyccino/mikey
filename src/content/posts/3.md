AI Note: Tailwind CSS is an open-source, "utility-first" CSS framework for rapidly building custom user interfaces.

The command line tool [Yo Code extension generator](https://www.npmjs.com/package/generator-code) will create all the files you need to build your theme.

1. Open up your terminal window and navigate to your main GitHub folder.
2. Then run on the terminal:

```
npm install -g yo generator-code
```

After the it finishes installing run:

```bash
yo code
```

The following files are generated by Yo Code
![files](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/u8h9v2s54lgcz16g33cu.png)

(The above links to an image to dispolay in the blog)

**1. .vscode/launch.json**

> The launch.json file is used to configure the debugger in Visual Studio Code.
> **2. themes/your-theme.json**
> This is the file you are going to be working on to create your theme. Right now it includes the styles for VS Code default theme.
> **3. .vscodeignore**
> A VSCODEIGNORE file is a text file used to exclude unnecessary files when packaging a Microsoft Visual Studio Code extension.
> **4. Changelog.md**
> You can write this changelog and if you publish it marketplace will link to it directly.
> **5. package.json**
> This file is where to pack, add and write some code to be able to publish it on the marketplace.
> **6. readme.md**
> The read me: edit before publishing.
> **7. vsc-extensions-quickstart.md**
> Tells you what all the files are but in short-hand.
